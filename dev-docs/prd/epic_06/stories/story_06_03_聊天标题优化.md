# Story 06-03: 聊天标题优化

> Epic: 好友列表与聊天标题优化
> Priority: Medium
> Points: 3

## 概述

优化聊天窗口标题区域的信息层次和交互体验。将好友名称左移，旁边用灰色小字显示好友描述简介，帮助用户快速回忆该好友的角色定位。同时增加点击好友名称弹出抽屉菜单的交互，提供更便捷的好友设置入口。

## 用户故事

### US-1: 查看好友描述
**作为** 用户，
**我希望** 在聊天标题区域看到好友的描述简介，
**以便** 我能快速回忆这个好友的角色定位，不用进入设置页面查看。

### US-2: 快速访问好友设置
**作为** 用户，
**我希望** 点击好友名称可以弹出设置菜单，
**以便** 我能方便地修改好友信息或进行其他操作，而无需离开聊天界面。

## 功能范围

### 包含
- 标题布局：好友名称左对齐，右侧显示灰色描述简介
- 文本处理：描述过长时截断并显示省略号，hover 时显示完整内容（tooltip）
- 交互增强：点击好友名称区域弹出右侧抽屉菜单（复用现有 `ChatDrawerMenu.vue`）
- 样式优化：确保标题区域符合微信风格设计

### 不包含（后续迭代）
- 标题区域的在线状态显示
- 好友头像点击交互
- 标题区域的自定义主题

## 验收标准

- [ ] AC-1: 聊天标题显示格式为 "好友名称 | 好友描述简介"
- [ ] AC-2: 好友名称使用主标题样式，描述使用灰色小字
- [ ] AC-3: 描述超过20个字符时截断显示 "..."
- [ ] AC-4: 鼠标悬停在截断的描述上时，显示完整描述（tooltip）
- [ ] AC-5: 点击好友名称或描述区域，从右侧弹出好友菜单抽屉
- [ ] AC-6: 界面响应式适配，窄屏时描述可隐藏
