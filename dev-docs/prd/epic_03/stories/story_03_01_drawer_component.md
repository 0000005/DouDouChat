# Story 03-01: 基础抽屉组件和菜单框架

> Epic: 聊天详情抽屉菜单  
> Priority: High  
> Points: 3

## 概述
实现从右侧滑入的抽屉面板及菜单项框架。用户点击聊天界面右上角的三点按钮后，抽屉从右侧滑入，顶部显示当前好友的头像和名称，下方展示 5 个菜单项。这是整个聊天详情功能的 UI 基础，后续 Story 将在此框架上添加具体功能逻辑。

## 用户故事

### US-1: 打开抽屉菜单
**作为** 用户，  
**我希望** 点击聊天界面右上角的三点按钮能够打开抽屉菜单，  
**以便** 快速访问当前会话的相关功能。

### US-2: 关闭抽屉菜单
**作为** 用户，  
**我希望** 点击抽屉外部区域或关闭按钮能够关闭抽屉，  
**以便** 回到正常的聊天界面。

### US-3: 查看菜单列表
**作为** 用户，  
**我希望** 在抽屉中看到清晰的菜单列表，包含会话列表、记忆列表、查找聊天记录、顶置聊天、清空聊天记录，  
**以便** 了解可以进行哪些操作。

## 功能范围

### 包含
- 抽屉组件的基础结构（使用 shadcn-vue 的 Sheet/Drawer 组件）
- 从右侧滑入/滑出的动画效果
- **顶部区域**：当前好友的头像、名称、添加好友按钮（占位）
- **菜单列表区域**：5 个菜单项的静态展示（图标 + 文字）
- "more-btn" 按钮的点击事件绑定
- 抽屉打开/关闭状态管理
- 抽屉关闭后视图状态重置（回到菜单列表）

### 不包含（后续迭代）
- 菜单项的具体功能实现（由后续 Story 实现）
- 添加好友按钮的实际功能（仅 UI 占位）
- 菜单项的权限控制或动态显示

## 验收标准

- [ ] AC-1: 点击 `ChatArea.vue` 中的 `more-btn` 按钮后，抽屉从右侧平滑滑入
- [ ] AC-2: 抽屉顶部显示当前好友的头像、名称、添加按钮（占位）
- [ ] AC-3: 抽屉中显示 5 个菜单项，每个菜单项包含适当的图标和文字标签
- [ ] AC-4: "清空聊天记录"菜单项使用红色文字（危险操作标识）
- [ ] AC-5: 点击抽屉外的遮罩层或关闭按钮后，抽屉平滑滑出关闭
- [ ] AC-6: 抽屉关闭后再次打开时，默认显示菜单列表（重置内部视图状态）
- [ ] AC-7: 抽屉的视觉风格严格参考微信聊天详情页（配色、间距、字体大小）
- [ ] AC-8: 抽屉宽度在桌面端为 360px，在移动端为全屏宽度
- [ ] AC-9: 菜单项点击有视觉反馈（hover 效果），但暂不执行具体功能
