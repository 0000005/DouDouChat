# Story 05-02: 好友库完整界面 (Gallery Full UI)

> Epic: 好友库 (Friend Gallery)
> Priority: High
> Points: 8

## 概述
【合并原 05-02 & 05-03】一次性构建好友库的完整前端体验，包括全局入口、页面布局、卡片网格、搜索筛选、以及点击查看详情的交互。

## 用户故事

### US-1: 全局入口访问
**作为** 用户，
**我希望** 通过侧边栏图标或新增按钮的下拉菜单，
**以便** 在任何时候都能快速打开好友库。

### US-2: 沉浸式浏览体验
**作为** 用户，
**我希望** 看到一个分类清晰、布局整齐的好友卡片墙，
**以便** 探索感兴趣的 AI 角色。

### US-3: 精准查找与深度了解
**作为** 用户，
**我希望** 能搜索关键词并点击卡片查看详细设定，
**以便** 快速找到并确认心仪的 AI 好友。

## 功能范围

### 包含
- **双入口实现**：
  - `IconSidebar` 新增"好友库"Tab 图标。
  - `Sidebar` 左上角"新增"按钮重构为下拉菜单（包含"手动创建"和"好友库"）。
- **页面布局**：创建 `FriendGallery.vue`。
  - 顶部：分类标签栏 (`Tabs`)，支持横向滚动。
  - 搜索框：支持 debounce 防抖，实时过滤。
  - 主体：响应式网格 (`Grid`) 展示好友卡片。
- **卡片组件**：展示头像、姓名、简介、标签，支持 Hover 效果。
- **详情展示**：
  - 点击卡片弹出 `Sheet` (侧边抽屉) 或 `Dialog`。
  - 展示大图、Markdown 格式简介。
  - **人格设定折叠面板**：默认折叠，点击展开查看完整的 System Prompt。
  - 底部显著的"添加好友"按钮（逻辑在 05-03）。
- **空状态处理**：搜索无结果时显示插画及"AI 创建"引导入口（链接至 05-05）。
- **对接 API**：调用 Story 05-01 的 `GET /api/friend-templates` 接口。

### 不包含
- 添加好友的后端调用逻辑（在 05-03）。
- AI 创建向导的具体实现（在 05-05）。

## 验收标准

- [ ] AC-1: 点击 IconSidebar 图标或下拉菜单项，主区域能正确切换显示好友库页面。
- [ ] AC-2: 标签栏默认选中"全部"，点击其他标签能触发 API 重新加载对应分类数据。
- [ ] AC-3: 卡片网格在宽屏显示多列，在窄屏自动调整为少列，无布局错乱。
- [ ] AC-4: 页面加载过程中有骨架屏 (Skeleton) 占位。
- [ ] AC-5: 在搜索框输入文本，列表能实时按需刷新。
- [ ] AC-6: 点击卡片弹出详情页，Markdown 渲染正常，人格设定支持折叠/展开。
- [ ] AC-7: 搜索无结果时，正确显示引导 UI。
