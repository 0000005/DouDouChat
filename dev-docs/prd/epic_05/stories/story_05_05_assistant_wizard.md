# Story 05-05: AI创建向导 (Assistant Wizard)

> Epic: 好友库 (Friend Gallery)
> Priority: High
> Points: 5

## 概述
前端交互实现。针对搜索不到心仪好友的用户，提供一个对话式的引导流程，让 AI 帮忙"捏"一个好友。

## 用户故事

### US-1: 无门槛创建
**作为** 用户，
**我希望** 既然没找到现成的，就能让 AI 帮我造一个，
**以便** 满足我特定的社交需求。

## 功能范围

### 包含
- **入口**：在 Story 05-02 的搜索空状态页接入。
- **对话式向导 UI**：
  - 简单的 Chat 界面或分步表单 `Stepper`。
  - 步骤 1：收集名称/身份。
  - 步骤 2：收集性格/细节。
  - 步骤 3：调用 05-04 的 `POST /api/friend-templates/generate` API，展示 Loading 动画。
- **结果预览与确认**：
  - 展示生成的卡片。
  - 提供"重新生成"和"确认添加"按钮。
  - 确认后调用 `POST /api/friend-templates/create-friend` 保存。

### 不包含
- 复杂的 prompt 手动微调。

## 验收标准

- [ ] AC-1: 从搜索无结果页能顺利进入向导。
- [ ] AC-2: 生成过程中的等待状态（可能长达 10s+）有友好的视觉反馈。
- [ ] AC-3: 用户可以不满意并重试，直到生成满意为止。
- [ ] AC-4: 最终保存的角色只归属于当前用户，不进入公共模版库。
