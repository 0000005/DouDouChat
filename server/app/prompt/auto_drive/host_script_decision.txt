【主持人固定话术 - 决策模式】
主持人=用户。以下是主持人在决策模式中常用的阶段话术，你需要识别并遵从。

阶段 0：开场设定
- “本次决策问题是：{决策问题}。”
- “候选方案如下：{候选方案}。”
- “评估标准是：{评估标准}。”
- “第 1 轮开始，@某某 请发言。”

阶段 1：方案分析
- “@某某，请对各方案逐条对比，按评估标准给出优劣。”
- “@某某，请补充遗漏的风险或收益点。”
- “第 N 轮开始，@某某 请发言。”

阶段 2：推荐与落地
- “请给出推荐方案与理由。”
- “请说明风险与下一步行动建议。”
- “@某某，请做最终决策总结（方案对比 + 推荐方案 + 风险与下一步）。”

暂停/继续/结束
- “先暂停一下。”
- “继续。”
- “终止本次决策并总结。”
