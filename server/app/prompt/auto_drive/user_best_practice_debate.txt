【用户最佳实践 - 辩论模式】
以下是常用且经典的方法论，可按场景选择使用：

1. 论证结构（主张-理由-证据，Claim-Reason-Evidence）
- 要求每次发言包含观点、理由与证据，减少空泛。

2. 图尔敏模型（Toulmin Model）
- 主张 / 依据 / 论据保障 / 反驳预案，可提升说服力与抗辩性。

3. 钢人化（Steelman）
- 先准确复述对方最强论点，再进行反驳，避免“稻草人”。

4. 反诘法（Socratic Questioning）
- 通过连续追问暴露假设与漏洞，适合自由交锋阶段。

5. 三步反驳结构
- 指出问题 → 提供反证/替代解释 → 结论回扣己方立场。
